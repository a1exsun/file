# Q1

## (a) Determine whether the clause is valid, satisfiable but not valid, or unsatisfiable.

### ϕ₁: *Satisfiable but not valid.*

> $ϕ₁ = ((A ∨ (B → C)) ∧ (¬C ∨ D)) → ((A → D) ∨ (C ↔ E))$

**Counterexample:**

- Take $A = T, D = F, C = F, E = T$ (and $B$ arbitrary)
- For antecedent $(A ∨ (B → C)) ∧ (¬C ∨ D)$:
    - $A$ is $T$, so $(A ∨ (B → C))$ is $T$.
    - $C$ is $F$, so $¬C$ is $T$, and $(¬C ∨ D)$ is $T$.
    - the antecedent is $T ∧ T$, which is **True**.
- For consequent $(A → D) ∨ (C ↔ E)$:
    - $A → D$ is $T → F$, which is $F$.
    - $C ↔ E$ is $F ↔ T$, which is $F$.
    - the consequent is $F ∨ F$, which is **False**.
- So $ϕ₁ ≡ T → F$, which is **False**. Hence $ϕ₁$ is not valid.

**Satisfiable:**

Set $D=T$, then $A → D$ is true, so the consequent is **True** ($T ∨ F$), making the whole formula to **True** ($T → T$). Hence $ϕ₁$ is **Satisfiable**.

### ϕ₂: *Satisfiable but not valid.*

> $ϕ₂ = (((¬A ∧ B) ∨ (C → D)) ∧ (¬E ∨ A)) → ((¬D ∨ F) ↔ (A ∧ B))$

**Counterexample:**

- Take $A = T, B = T, D = T, F = F$ (and $C, E$ arbitrary)
- For antecedent $((¬A ∧ B) ∨ (C → D)) ∧ (¬E ∨ A)$:
    - $A$ is $T$, so $¬A$ is $F$, and $(¬A ∧ B)$ is $F$.
    - $D$ is $T$, so $(C → D)$ is $T$.
    - $A$ is $T$, so $(¬E ∨ A)$ is $T$.

- For consequent $(¬D ∨ F) ↔ (A ∧ B)$:

    - $A$ is $T$, and $B$ is $T$, so $A ∧ B ≡ T ∧ T$, which is $T$.

**Satisfiable:**

Based on the counterexample, set $A = F,E = T$. Then $¬E ∨ A = F$, making the antecedent **False**, hence the whole implication is **True**. Therefore $ϕ₂$ is **Satisfiable**.

### ϕ₃: *Satisfiable but not valid.*

> $ϕ₃ = ((A ↔ B) ∨ (C → D)) ∧ ((¬A ∨ ¬D) → (F ∧ ¬G))$

**Counterexample:**

- Take $A = F, B = F, D = T, F = F, G = F$ (and $C$ arbitrary)
- For first conjunct $((A ↔ B) ∨ (C → D))$:
    - Since $A = F$ and $B = F$, so $A ↔ B = T$. 
    - Hence the first conjunct is **True**.
- For second conjunct $((¬A ∨ ¬D) → (F ∧ ¬G))$:
    - Since $A = F$, so $¬A = T$, then the antecedent $(¬A ∨ ¬D)$ is True.
    - $F ∧ ¬G = F ∧ T$, which is False.
    - Hence the second conjunct is $T → F$, which is **False**.
- The whole formula is $ϕ₃ ≡ T ∧ F$, which is **False**. Hence $ϕ₃$ is not valid.

**Satisfiable:**

- Based on the counterexample, set $A = T, D = T$.
- For the first conjunct:
    - Since $D = T$, $(C → D)$ is $T$, so $((A ↔ B) ∨ (C → D))$ is **True**.
- For the second conjunct:
    - $¬A ∨ ¬D ≡ F ∨ F$, which is False.
    - Thus $(¬A ∨ ¬D) → (F ∧ ¬G)$ has a False antecedent, hence is **True**.
- Thus the whole formula is **True**. Hence $ϕ₃$ is satisfiable.

## (b) Convert each formula into CNF. Identify Horn clauses where applicable.

### ϕ₁

> $ϕ₁ = ((A ∨ (B → C)) ∧ (¬C ∨ D)) → ((A → D) ∨ (C ↔ E))$

- **Eliminate implications and equivalences**
    - $B → C ≡ (¬B ∨ C)$
    - $A → D ≡ (¬A ∨ D)$
    - $C ↔ E ≡ (¬C ∨ E) ∧ (¬E ∨ C)$
    - So the formula is $((A ∨ (¬B ∨ C)) ∧ (¬C ∨ D)) → ((¬A ∨ D) ∨ ((¬C ∨ E) ∧ (¬E ∨ C)))$
- **Replace the Outer Implication**
    - Take the whole formula as $X → Y$, then $X → Y ≡ (¬X ∨ Y)$
    - The formula becomes: $¬((A ∨ ¬B ∨ C) ∧ (¬C ∨ D)) ∨ ((¬A ∨ D) ∨ ((¬C ∨ E) ∧ (¬E ∨ C)))$ 
- **Apply De Morgan's Law to the left part**
    - Left is $¬(X ∧ Y)$, where $X = A ∨ ¬B ∨ C$ and $Y = ¬C ∨ D$.
    - $¬(X ∧ Y) ≡ ¬X ∨ ¬Y$
    - $¬X = ¬(A ∨ ¬B ∨ C) ≡ ¬A ∧ ¬(¬B) ∧ ¬C ≡ ¬A ∧ B ∧ ¬C$ (De Morgan's Law)
    - $¬Y = ¬(¬C ∨ D) ≡ ¬(¬C) ∧ ¬D ≡ C ∧ ¬D$ (De Morgan's Law)
    - Thus, the left part becomes: $(¬A ∧ B ∧ ¬C) ∨ (C ∧ ¬D)$
 - **First distribute the inner conjunction on the right**
    - $((¬A ∨ D) ∨ ((¬C ∨ E) ∧ (¬E ∨ C))) ≡ (¬A ∨ D ∨ ¬C ∨ E) ∧ (¬A ∨ D ∨ ¬E ∨ C)$
    - Let the two clauses be:
        - $S₁ = (¬A ∨ D ∨ ¬C ∨ E)$
        - $S₂ = (¬A ∨ D ∨ ¬E ∨ C)$
    - Thus, the right part becomes: $S₁ ∧ S₂$
 - **The whole formula becomes the conjunction of two disjunctions**
    - $[(¬A ∧ B ∧ ¬C) ∨ (C ∧ ¬D) ∨ S₁] ∧ [(¬A ∧ B ∧ ¬C) ∨ (C ∧ ¬D) ∨ S₂]$
 - **Distribute and simplify the first term (CNF-compatible)**
    - $(C ∧ ¬D) ∨ S₁ ≡ (C ∨ S₁) ∧ (¬D ∨ S₁)$
    - But $C ∨ S₁ = C ∨ (¬A ∨ D ∨ ¬C ∨ E)$ contains $C ∨ ¬C$, which is a tautology; $¬D ∨ S₁$ contains $¬D ∨ D$, also a tautology.
    - Therefore, $(C ∧ ¬D) ∨ S₁$ is a tautology, hence
    - $[(¬A ∧ B ∧ ¬C) ∨ (C ∧ ¬D) ∨ S₁]$ is a tautology
    - This whole conjunct can be dropped
 - **Distribute and simplify the second term**
    - $(C ∧ ¬D) ∨ S₂ ≡ (C ∨ S₂) ∧ (¬D ∨ S₂)$
    - Here $C ∨ S₂ = S₂$ (because $S₂$ already contains $C$), and $¬D ∨ S₂$ contains $¬D ∨ D$, which is a tautology.
    - Therefore, $(C ∧ ¬D) ∨ S₂ ≡ S₂$, hence
    - $[(¬A ∧ B ∧ ¬C) ∨ S₂] ≡ (¬A ∨ S₂) ∧ (B ∨ S₂) ∧ (¬C ∨ S₂)$, where:
        - $¬A ∨ S₂ = S₂$ (since $S₂$ already contains $¬A$)
        - $¬C ∨ S₂ = (¬C ∨ ¬A ∨ D ∨ ¬E ∨ C)$ contains $¬C ∨ C$, a tautology, so it can be removed
        - The remaining $(B ∨ S₂)$ is a superset clause compared to $S₂$ (weaker), and is redundant in CNF, so it can be removed
 - **Final CNF**
  $(¬A ∨ C ∨ D ∨ ¬E)$
 - **Horn check**
  The clause $¬A ∨ C ∨ D ∨ ¬E$ contains two positive literals $D$ and $C$ (more than one positive literal), therefore it is not a Horn clause, hence ϕ₁ is not a Horn formula.

### ϕ₂

    - Second conjunct: $([D ∧ ¬F] ∨ [A ∧ B])$

    $Y ≡ (D ∨ A) ∧ (D ∨ B) ∧ (¬F ∨ A) ∧ (¬F ∨ B) ∧ (¬D ∨ F ∨ ¬A ∨ ¬B)$

    $X ≡ [ (¬A ∧ B) ∨ (¬C ∨ D) ] ∧ (¬E ∨ A)$
- **Replace the outer implication**
  - The formula is $X → Y ≡ (¬X ∨ Y)$
- **Negate the left part and simplify**
  - Let $X1 = ( (¬A ∧ B) ∨ (¬C ∨ D) )$, $X2 = (¬E ∨ A)$

  - $¬X1 = ¬( (¬A ∧ B) ∨ (¬C ∨ D) ) ≡ ¬(¬A ∧ B) ∧ ¬(¬C ∨ D)$  
    $≡ (A ∨ ¬B) ∧ (C ∧ ¬D)$
  - So $¬X ≡ [ (A ∨ ¬B) ∧ C ∧ ¬D ] ∨ [ E ∧ ¬A ]$
- **Combine into $(¬X ∨ Y)$**
  - Using distribution $(P ∨ (Y_1 ∧ … ∧ Y_n)) ≡ (P ∨ Y_1) ∧ … ∧ (P ∨ Y_n)$  
    Set $P := ¬X$, expand over each clause of $Y$
  - After simplification, tautology elimination, and subsumption removal, the final CNF is obtained
- **Final CNF**
  $(A ∨ C ∨ D ∨ E) ∧  
   (A ∨ D ∨ E ∨ ¬B) ∧  
   (B ∨ C ∨ D ∨ ¬A) ∧
